# Thyroid Disease Detection

This repository contains the implementation code for the manuscript:

**A High-Accuracy Stacking Ensemble Model for Precise Thyroid Disease Detection with SHAP and LIME Explanations**

## Contents
- `Thyroid_Code.ipynb`: main notebook containing preprocessing, feature selection, model training, evaluation, and interpretability analysis

## Main components
- Data preprocessing
- Class imbalance handling using SMOTE
- Consensus-based feature selection
- Baseline machine learning models
- Proposed stacking ensemble model
- SHAP and LIME explanations
- Stratified k-fold cross-validation
- Statistical significance analysis

## Requirements
Recommended environment:
- Python 3.x
- Jupyter Notebook

Main libraries used:
- numpy
- pandas
- scikit-learn
- xgboost
- imbalanced-learn
- shap
- lime
- matplotlib
- seaborn
- scipy

## Usage
1. Open `Thyroid_Code.ipynb`
2. Run the cells sequentially
3. Update dataset paths if needed before execution

## Dataset
The study uses the publicly available thyroid dataset from:
- UCI Machine Learning Repository
- Kaggle mirror: https://www.kaggle.com/datasets/jayaprakashpondy/thyroid-dataset
